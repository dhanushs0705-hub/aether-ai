# AETHER AI

Welcome to **AETHER AI**, your Holographic Cyberpunk Life OS. 

This is a premium, cinematic full-stack web application that acts as a personal digital universe for productivity, goals, habits, learning, wellness, and intelligent life management.

## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the holographic environment:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:3000`

## 🔮 Features

- **Cinematic Landing Page:** Ambient particles, glowing gradients, and AI boot sequences.
- **Holographic Dashboard:** Glassmorphism UI, floating widgets, and dynamic active quests.
- **AI Life Assistant:** Simulated neural link interface with typing animations.
- **Focus Mode:** Immersive Pomodoro timer with ambient pulsing effects.
- **Behavioral Tracking:** Habit streaks with flame animations.
- **Trajectory Management:** Visual roadmaps for your prime directives.
- **Neural Logs:** Journaling with AI sentiment analysis simulation.
- **Evolution Path:** XP and leveling system for gamified progression.

## 🛠 Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Custom CSS (Glassmorphism, Neon FX)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Audio:** Web Audio API (Synthesized Sci-Fi UI Sounds)
